# Jump Campaign Builder - Vision Prototype

A vision prototype showcasing the future of AI-powered campaign creation for the Jump ticketing platform.

## Overview

This prototype demonstrates a fully autonomous campaign builder that could eliminate traditional ticketing operations roles by handling everything from strategy to execution.

**Vision Timeline:** 5 years from now

**Philosophy:** AI does 80%, user reviews 15%, decides 5%

## Features (Current Implementation)

- Campaign dashboard with key metrics
- Active campaign overview
- Real-time analytics display
- Modern, intuitive UI matching professional ticketing platforms

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI:** Custom components

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the prototype.

### Build

```bash
npm run build
npm start
```

## Project Structure

```
/app
  ├── layout.tsx      # Main layout with sidebar and navigation
  ├── page.tsx        # Campaigns dashboard
  └── globals.css     # Global styles and Tailwind imports
```
